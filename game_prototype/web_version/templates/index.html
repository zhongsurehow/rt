游戏规则

基本概念

天机变是一款融合易经哲学的策略卡牌游戏。玩家通过打出卦象卡牌、平衡阴阳五行来获得胜利。

玩家属性

  • 气 (Qi)：生命力，影响行动能力
  • 道行 (Dao)：修为，影响卡牌威力
  • 诚意 (Sincerity)：真诚度，影响学习效果
  • 阴阳：需要保持平衡，影响特殊能力
  • 五行：木火土金水，相生相克

胜利条件

通过合理运用卦象、维持阴阳平衡、掌握五行相生相克关系来击败对手。

📖 天机变游戏规则

🎯 游戏目标

在八卦阵中通过策略布局，率先达到以下任一胜利条件：

  • 道行胜利：道行值达到100点
  • 分数胜利：总分数达到50分
  • 控制胜利：同时控制4个或以上八卦位置

🎴 基本玩法

  1. 选择位置：点击八卦位置移动到该位置
  2. 使用卡牌：选择手牌中的卡牌使用
  3. 管理资源：平衡阴气、阳气和气力
  4. 策略布局：利用五行相生相克制定策略

🎓 新手教程

第一步：了解界面

  • 🎯 八卦阵：游戏的主要区域，点击移动到不同位置
  • 🃏 手牌区：显示你的卡牌，点击使用
  • 📊 状态栏：显示阴气、阳气、气力和道行
  • 📝 日志区：显示游戏进程和提示

第二步：基本操作

  1. 1️⃣ 移动：点击八卦位置移动角色
  2. 2️⃣ 使用卡牌：点击手牌中的卡牌
  3. 3️⃣ 查看效果：观察资源和分数变化
  4. 4️⃣ 结束回合：点击"结束回合"按钮

第三步：策略要点

  • 💡 平衡资源：保持阴阳平衡，避免资源枯竭
  • 🔄 利用五行：使用相生相克关系获得优势
  • 🎯 控制位置：占据关键八卦位置
  • 📈 积累道行：通过智慧行动提升道行值
